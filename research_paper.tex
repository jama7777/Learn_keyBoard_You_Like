\begin{abstract}
Proficiency in touch typing remains a critical competency in the digital era, yet traditional typing tutors often rely on repetitive, static content that fails to maintain user engagement. This paper introduces TypeMaster AI, an advanced multimodal typing tutor that leverages Large Language Models (LLMs) to provide personalized and interactive learning experiences. By integrating the Gemini 2.0 API and OpenRouter, the system offers dynamic lesson generation based on user-defined topics, real-time grammar correction for custom inputs, and an immersive ``Melody Typing'' mode featuring predefined classical and contemporary music. Our results demonstrate that the integration of diverse AI modalities and auditory feedback significantly enhances user motivation and provides a more versatile training environment compared to conventional methods.
\end{abstract}

\section{Introduction}
Touch typing is no longer a specialized skill but a fundamental requirement for productivity across various professional fields. Despite its importance, the pedagogical approach to typing hasn't evolved significantly in decades. Most contemporary instructors still utilize fixed string sets or randomized character sequences, which can become monotonous. 

TypeMaster AI aims to revolutionize this domain by treating typing not just as a motor skill, but as an interactive experience. By utilizing state-of-the-art AI, we enable users to practice using content they are actually interested in—whether it's news about technology, snippets of Python code, or the melodies of iconic musical compositions.

\subsection{Frontend and UI Design}
The application is developed using React 19 and Vite. Key components include:
\begin{itemize}
    \item \textbf{Virtual Keyboard:} Real-time finger positioning hints using character-to-key mapping.
    \item \textbf{Dynamic Typing Area:} Precise tracking of WPM, Accuracy, and Mistakes.
    \item \textbf{History Service:} Local persistence of user progress and search history.
\end{itemize}

\subsection{AI Service Layer}
The core intelligence is provided by the Gemini 2.0 model. We implemented a robust gateway that utilizes OpenRouter as a fallback.
\begin{itemize}
    \item \textbf{Lesson Generation:} Dynamic creation of exercises based on user topics (e.g., ``Quantum Computing'').
    \item \textbf{Grammar Correction:} Automatic refinement of user-pasted text to ensure high-quality practice material.
\end{itemize}

\section{Immersive Audio Feedback}
\subsection{Melody Typing Mode}
TypeMaster AI introduces an innovative ``Melody Typing'' system. Unlike traditional tutors that use generic click sounds, this system maps each correct keystroke to the next note of a musical melody.

\subsection{Predefined Musical Library}
The system includes an extensive library of predefined classic and contemporary tunes, including:
\begin{itemize}
    \item \textbf{Classical:} Fur Elise, Ode to Joy, Canon in D.
    \item \textbf{Pop Culture:} Super Mario Theme, Tetris, Star Wars, Harry Potter.
    \item \textbf{Educational:} Twinkle Twinkle Little Star, London Bridge.
\end{itemize}
This auditory feedback loop creates a gamified experience where the user ``performs'' the music through accurate typing.
